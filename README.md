# BACKGAMMON 
The classic oriental game, console written using C++, STL. 
- 2 Modes: 
  - Player vs Player
  - Player vs AI
- Options:
  - Includes All Blocking Rules
  - Saving Current Game
  - Loading Last Game
  - Restart Game
- Gamepad:
  - Arrows - move figure
  - Enter - confirm moving/action
  - Esc - cancel moving/action
